# Be That Space - Luxury Minimalistic Website

A luxury, minimalistic website for Be That Space, an international architecture and design firm. This website embodies the philosophy of "abstract manifested" through clean design, elegant typography, and smooth interactions.

## Features

- **Luxury Minimalistic Design**: Clean, sophisticated aesthetic with premium typography
- **Fully Responsive**: Optimized for all devices and screen sizes
- **Smooth Animations**: Elegant fade-in and scroll animations
- **Modern Typography**: Playfair Display for headings, Inter for body text
- **Accessible**: Semantic HTML and keyboard navigation support
- **Performance Optimized**: Lightweight and fast-loading

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --color-primary: #000000;
    --color-secondary: #ffffff;
    --color-accent: #d4af37;
    --color-text-light: #666666;
    --color-text-dark: #1a1a1a;
}
```

### Typography
Font families can be changed in the `:root` variables or in the HTML `<head>` section.

## Development

Simply open `index.html` in a web browser. No build process or dependencies required.
